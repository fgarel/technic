Premiere Diapo
==============

Une documentation, ecrite au format Restful, peut être transformée en plusieurs formats de sortie.

Seconde Diapo
=============

Les formats qui peuvent être générés sont :

  * html
  * pdf
  * presentation (hieroglyph)
  * presentation (hovercraft)
